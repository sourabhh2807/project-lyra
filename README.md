## What This Is

Project Lyra is a self-evolving, zero-cost autonomous content pipeline
that runs entirely inside GitHub's free infrastructure.

It treats every video as a living genetic organism — born from research,
tested by the market, selected for fitness, mutated for superiority,
and cloned at scale. No human in the loop after initial setup.
